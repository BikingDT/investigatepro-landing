## Personalization Tips

1. **Research before sending:** Check LinkedIn for recent posts, company news
2. **Industry specifics:** Adjust examples for mining/construction/manufacturing
3. **Company size:** Adjust ROI numbers based on employee count
4. **Recent events:** Reference any public incidents (carefully, respectfully)
5. **Mutual connections:** Mention if you have any shared connections

---

## Reply Handling

**Positive reply:** Move to demo scheduling sequence
**"Not now":** Add to nurture sequence, follow up in 90 days
**"Not interested":** Thank them, ask for referral, add to newsletter only
**"Unsubscribe":** Remove immediately, mark as do-not-contact
