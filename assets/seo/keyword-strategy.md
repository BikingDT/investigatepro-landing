## On-Page SEO Checklist

### Every Page
- [ ] Target keyword in title tag (front-loaded)
- [ ] Target keyword in H1
- [ ] Target keyword in first 100 words
- [ ] Target keyword in URL slug
- [ ] Meta description with keyword (155 chars)
- [ ] 2-3 internal links to related content
- [ ] 1-2 external links to authoritative sources
- [ ] Alt text on all images
- [ ] Schema markup (Article, Product, FAQ as appropriate)

### Blog Posts
- [ ] Target keyword in title
- [ ] Related keywords in H2s
- [ ] Table of contents for 1,500+ word posts
- [ ] Featured image with alt text
- [ ] Author byline with credentials
- [ ] Publish date visible
- [ ] CTA to trial/demo

## Technical SEO Checklist

### Site Structure
- [ ] XML sitemap submitted to Google Search Console
- [ ] Robots.txt configured correctly
- [ ] Canonical URLs on all pages
- [ ] Mobile-responsive design
- [ ] Page speed < 3 seconds
- [ ] HTTPS everywhere

### Schema Markup
- [ ] Organization schema on homepage
- [ ] Product schema on pricing page
- [ ] Article schema on blog posts
- [ ] FAQ schema on FAQ page
- [ ] BreadcrumbList on all pages

### Indexing
- [ ] Google Search Console configured
- [ ] Bing Webmaster Tools configured
- [ ] Core pages indexed
- [ ] No duplicate content issues
- [ ] No broken links (404s)

## Link Building Strategy

### Quick Wins
- **Safety directories:** Submit to safety software directories
- **Industry associations:** Get listed with safety associations

### Content-Driven Links
- **Original research:** Survey on investigation practices
- **Free tools:** ROI calculator, PEEPO worksheet (linkable assets)
- **Infographics:** Hierarchy of Controls, Investigation Process
- **Expert roundups:** "Safety leaders on investigation best practices"

### Partnership Links
- **Consultant partners:** Links from their sites
- **Integration partners:** Links from complementary software
- **Training providers:** Links from ICAM training organizations
